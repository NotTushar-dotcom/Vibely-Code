# Vibely Code

A modern, aesthetic blog website dedicated to web development tutorials, AI insights, and creative tech content. Built with React and styled with Tailwind CSS for a sleek, responsive experience.

- **Newsletter Subscription**: Stay updated with latest tutorials
- **Social Media Sharing**: Share articles on Facebook, Twitter, and Google+
- **Responsive Design**: Optimized for all devices
- **Search Functionality**: Find blogs quickly
- **Animated UI**: Smooth transitions with Framer Motion

   ```bash
   npm run dev
   ```
